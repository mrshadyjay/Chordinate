# Chordinate
A native Android app for guitarists who want to break out of 
creative block and try different chord progressions — built as one of my final year Computer Science 
projects at Birmingham City University.


## ⬇️ Download
1. Download the APK to your Android phone
2. Go to **Settings → Security → Enable "Install unknown apps"**
3. Open the file and tap **Install**

> Requires Android 7.0 or above


## What it does
Pick a key, mode and mood — Chordinate generates a chord 
progression instantly. Tap any chord to hear it, save 
progressions you like, and practise with the built-in 
metronome.

## Tech Stack
Kotlin · MVVM · Room · AudioTrack · Canvas API · 
Jetpack Navigation · LiveData · Kotlin Coroutines
